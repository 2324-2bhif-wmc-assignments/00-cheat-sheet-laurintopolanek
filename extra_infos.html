Process of an HTTP request

  1. Request to DNS
  2. DNS responds IP
  3. Request to Web Server
  4. Web Server checks if the request is valid
  5. If request is valid, Web Server responds HTML, CSS, ....
    If not, the Web Server responds a HTML Status Code.
